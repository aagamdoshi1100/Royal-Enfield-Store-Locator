# Royal Enfield Store Locator

This project is a store locator application built with Next.js, designed to help users find Royal Enfield motorcycle dealerships and service centers based on their location or selected criteria.

## Features

- **Google Maps Integration**: Displays store locations on an interactive map.
- **Store Details**: Provides detailed information about each store, including address, contact information, and operating hours.
- **State and City Filtering**: Users can filter stores by state and city to find the nearest Royal Enfield showroom or service center.
- **Custom Scrollbars**: Styled vertical scrollbars horizontal scrollbars for a better user experience.

## Technologies Used

- **Next.js**: Server-side rendering and static site generation framework.
- **React**: JavaScript library for building user interfaces.
- **Google Maps JavaScript API**: For rendering maps and markers based on the store locations.
- **MongoDB**: Database for storing showroom information.
- **Mongoose**: ODM for MongoDB.

## Setup Instructions

### Prerequisites

- Node.js (>= 14.x)
- MongoDB instance (local or cloud-based)
- Google Maps API key
